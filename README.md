Relay is a secure messaging app, with features like end-to-end encryption, biometric authentication and more. It is built using Flutter & Firebase.

## Thanks for auditing!
# 3 Ways to audit

1. Watch the [video](https://photos.app.goo.gl/Pgk82arY9qxmLfUEA) and see other audit answers below.
2. Use the android app link to download the app and run on your own device
3. Clone the repo and run the project with flutter run ( for debug mode ) or flutter run --release ( for release mode). Requires flutter to be installed.
